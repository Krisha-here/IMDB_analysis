# IMDB Dataset Analysis

This repository contains my submission for the IMDB Dataset Analysis task, where data preprocessing, exploratory data analysis (EDA), and visualization were performed on the IMDB movie reviews dataset.

---

## 📘 Project Overview

The objective of this project is to analyze the IMDB movie reviews dataset and extract meaningful insights through preprocessing, statistical analysis, and visualizations.

---

## 📂 Dataset

The dataset contains movie reviews categorized into:

* Positive
* Negative

---

## 🔧 Steps Performed

### Data Preprocessing
* Loaded the dataset using Pandas
* Inspected dataset structure and data types
* Checked for missing values
* Prepared data for analysis
### Exploratory Data Analysis (EDA)
* Analyzed sentiment distribution
* Examined dataset characteristics
* Generated summary statistics
### Visualization
* Created bar charts to visualize sentiment distribution
* Used Matplotlib for graphical representation of findings

---

## 📊 Key Findings

* The dataset contains both positive and negative movie reviews.
* No missing values were found in the dataset.
* The dataset is balanced, containing an equal number of positive and negative reviews.
* Visualizations provided a clear understanding of sentiment distribution.
